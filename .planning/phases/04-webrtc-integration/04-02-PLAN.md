## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| client → TURN credential endpoint | Untrusted client requests TURN credentials |
| TURN server → Internet | Public relay server, potential bandwidth theft target |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-04-06 | Tampering | TURN credential generation | mitigate | Require authenticated JWT token before generating credentials (verify in Socket.IO middleware) |
| T-04-07 | Information Disclosure | TURN credential leakage | mitigate | Use short TTL (1 hour) for credentials, HMAC-SHA1 signature prevents forgery |
| T-04-08 | Denial of Service | TURN bandwidth exhaustion | mitigate | Rate limit credential generation (max 1 per minute per user), monitor bandwidth usage |
| T-04-09 | Spoofing | TURN server impersonation | mitigate | Use TURN over TLS (turns: scheme), validate certificates |
| T-04-10 | Tampering | ICE candidate injection | mitigate | Validate ICE parameters with Zod schema before creating transport |
</threat_model>
