    - Accept parameters: sessionStartTime (Date), durationMinutes (number, default 45)
    - State: isPromptActive (boolean, default false), timeRemaining (number, default 0)
    - Calculate endTime = addMinutes(sessionStartTime, durationMinutes)
    - useEffect with interval every 1000ms:
      - Calculate remaining = differenceInSeconds(endTime, Date.now())
      - Calculate minutesRemaining = differenceInMinutes(endTime, Date.now())
      - Update timeRemaining to Math.max(0, remaining)
      - Set isPromptActive=true when minutesRemaining <= 5 AND minutesRemaining >= 0
      - Set isPromptActive=false otherwise
      - Clear interval when remaining <= 0
    - Return { isPromptActive, timeRemaining, minutesRemaining }
  </action>
